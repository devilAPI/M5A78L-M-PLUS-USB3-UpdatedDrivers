Realtek FE/GbEファミリーコントローラーをご使用頂き誠に有難うございます。


紹介

この取扱説明書にはRealtek FE/GbEファミリーコントローラーのインストール情報、インストール中に発生するかもしれない問題、トラブルシューティングが掲載されています。また、詳細プロパティの設定方法も含まれています。


本書の使用方法

本章では、以下のWindowsオペレーティングシステムに「Realtek FE/GbEファミリーコントローラー」をインストールする情報が提供されています：

Windows 7
  • Windows 7 (32-bit) 用 NDIS 6.20 ミニポートドライバ
  • Windows 7 (64-bit) 用 NDIS 6.20 ミニポートドライバ
